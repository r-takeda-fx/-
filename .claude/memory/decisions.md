# 意思決定記録（Decisions）

プロジェクトの重要な技術的・設計的決定を記録します。

---

## D1: ホスティング先の選定

**日付**: 2026-01-27
**ステータス**: 承認済み

### コンテキスト
無料でWebアプリをホスティングする必要がある

### 決定内容
GitHub Pages を使用する

### 理由
- 完全無料
- 静的サイトに最適
- GitHubと連携しやすい
- カスタムドメイン対応可能

### 影響
- 静的サイトとして構築する必要あり
- サーバーサイド処理は使用不可

## D2: フロントエンド技術選定

**日付**: 2026-01-27
**ステータス**: 承認済み

### コンテキスト
シンプルで軽量なWebアプリを構築したい

### 決定内容
Vanilla JavaScript + CSS3 を使用（フレームワークなし）

### 理由
- 学習コスト低
- 軽量で高速
- 依存関係なし
- 保守が容易

### 影響
- 複雑な状態管理は手動で実装
- コンポーネント再利用は限定的
