# consulting-ai-news 設計書

**作成日**: 2026-01-27
**ステータス**: 承認済み

---

## 1. 目的（Why）

コンサル業界とAIの最新情報を**1箇所で効率的に収集**し、毎日の情報収集時間を短縮する

## 2. ゴール（What）

### MVP（最小構成）
- タブ切り替え式ニュースダイジェストWebアプリ
- 2つのタブ：「コンサル業界」「AI」
- スマホ対応（レスポンシブデザイン）
- 無料ホスティング（GitHub Pages）

### Nice to Have（後回し）
- お気に入り保存機能
- 検索・フィルター機能
- プッシュ通知

## 3. 心理ゴール（Who）

| Before | After |
|--------|-------|
| 毎日複数サイトを巡回、情報収集に時間がかかる | 1アプリで完結、通勤中にサクッとチェック |
| 重要ニュースを見逃すことがある | カテゴリ別に整理され、漏れなく確認できる |

## 4. 想定障害と対策

| 障害 | 対策 |
|------|------|
| 無料でホスティングしたい | GitHub Pages（完全無料） |
| ニュースデータの取得 | RSS / News API無料枠 / 静的JSON更新 |
| スマホ対応 | CSS Flexbox/Gridでレスポンシブ実装 |

## 5. アクションプラン

1. HTML/CSS基本構造（タブUI + レスポンシブ）
2. JavaScriptでタブ切り替え機能
3. ニュースデータ取得・表示機能
4. GitHub Pagesへデプロイ
